📌 หมายเหตุ
=======================================================

กรณีที่ตรวจสอบแต่ไม่พบในข้อมูล existing-songs.json:
- "ดา เอ็นโดรฟิน" vs "เอ็นโดรฟิน" - ไม่พบในไฟล์
- "ลิปตา (Lipta)" - ไม่พบในไฟล์ (มีแต่ Lipta เท่านั้น)
- "ตั้ว (วงสไมล์บัฟฟาโล่)" - ไม่พบในไฟล์
- "พกส่ (Pause)" vs "Pause" - ไม่พบในไฟล์

กรณีเหล่านี้อาจอยู่ในฐานข้อมูล Supabase แต่ยังไม่ได้ export มาในไฟล์ existing-songs.json

หากต้องการแก้ไขในฐานข้อมูล Supabase โดยตรง สามารถใช้ SQL:

UPDATE band_songs 
SET artist = 'Tattoo Colour' 
WHERE artist = 'TATTOO COLOUR';

UPDATE band_songs 
SET artist = 'COLORPiTCH' 
WHERE artist = 'Colorpitch';
