📝 สมัครสมาชิก

  1. กรอกรหัสวง (6 หลัก) — ขอจากผู้จัดการวงของคุณ ระบบจะแสดงชื่อวงเพื่อยืนยัน
  2. กรอกข้อมูลส่วนตัว — คำนำหน้า ชื่อ นามสกุล ชื่อเล่น และเลือกตำแหน่ง/เครื่องดนตรี
  3. สร้างบัญชี — กรอกอีเมล ตั้งรหัสผ่าน (8 ตัวขึ้นไป) ตรวจสอบข้อมูลแล้วกด ยืนยัน
💡 สร้างวงใหม่? ไปที่ลิงก์ "ขอสร้างวงใหม่" ในหน้าล็อกอิน กรอกข้อมูลวง + ผู้จัดการ → รอแอดมินอนุมัติ (ไม่เกิน 24 ชม.) → ได้รหัสวงแจกสมาชิก

🔐 เข้าสู่ระบบ

  1. เปิดเว็บแอป BandThai → หน้าล็อกอิน
  2. กรอก อีเมล และ รหัสผ่าน ที่สมัครไว้
  3. กด เข้าสู่ระบบ → ไปหน้าแดชบอร์ด
💡 ลืมรหัสผ่าน? กด "ลืมรหัสผ่าน" แล้วกรอกอีเมลเพื่อรับลิงก์รีเซ็ต ❘ ลืมอีเมล? กด "ลืมอีเมล" แล้วใช้ชื่อเล่นหรือเบอร์โทรค้นหา
⚠️ เปิดจาก LINE/Facebook ในตัว? แนะนำให้กดเปิดใน Safari/Chrome เพื่อประสบการณ์ที่ดีที่สุด

🏠 แดชบอร์ด

หน้าแรกหลังล็อกอิน รวมข้อมูลสำคัญทุกอย่างไว้ในที่เดียว:

  • ⏰ ลงเวลาด่วน — เลือกวันที่ สถานที่ รอบเวลา แล้วกดยืนยัน (ลาได้ พร้อมระบุคนแทน)
  • 📊 สรุปรายได้ — แสดง tabs สัปดาห์/เดือน/ปี พร้อมชั่วโมงและรายได้
  • 📅 งานที่จะมา — งานปกติ + งานนอก ที่ใกล้ถึง
  • 🎤 งานนอก — แสดงงานนอกถัดไปพร้อมรายละเอียด
  • 📋 เพลย์ลิสต์วันนี้ — ลิสต์เพลงของวันนี้ (ถ้ามี)
  • ⚡ ทางลัด — 6 ปุ่มเข้าถึงหน้าสำคัญอย่างรวดเร็ว
💡 ผู้จัดการวงจะเห็นเพิ่มเติม: รายชื่อสมาชิกรอการอนุมัติ + Admin Online Panel แสดงคนที่ออนไลน์อยู่

🎼 คลังเพลง

รวมเพลงทั้งหมดของวง แสดงเป็นตาราง Grid พร้อมระบบค้นหาและกรอง

  • 🔍 ค้นหา — พิมพ์ชื่อเพลง/ศิลปิน ค้นหาแบบ real-time
  • 🏷️ ตัวกรอง — กรองตามคีย์ แนวเพลง นักร้อง อารมณ์ ยุค ฯลฯ
  • การ์ดเพลง — แสดงชื่อเพลง ศิลปิน คีย์ แนวเพลง ประเภทนักร้อง (ชาย🔵/หญิง🩷/คู่🟢)
  • + เพิ่มเพลง — กดปุ่มเพื่อไปหน้าเพิ่มเพลงใหม่
💡 มีเพลงจำนวนมาก? ใช้ "เพิ่มจากคลังกลาง" เพื่อเลือกหลายเพลงจากฐานข้อมูลกลางพร้อมกัน ง่ายกว่าเพิ่มทีละเพลง!

➕ เพิ่ม/แก้ไขเพลง

  1. กรอก ชื่อเพลง (จำเป็น) และ ศิลปิน (autocomplete)
  2. เลือก คีย์ (C/Am, 1#-7#, 1b-7b) และกรอก BPM
  3. เลือก ยุค (80s-2020s), อารมณ์เพลง (มัน/หวาน/เศร้า/นิ่ง/ฮึกเหิม) และ แนวเพลง
  4. เลือก ประเภทนักร้อง — ชาย 🔵 / หญิง 🩷 / คู่ 🟢
  5. กด 💾 บันทึก
💡 กดปุ่ม 🤖 AI Auto-Tag เพื่อให้ AI ช่วยกรอกข้อมูล (คีย์, BPM, แนวเพลง, อารมณ์, ยุค) ให้อัตโนมัติ!

📋 เซ็ตลิสต์

สร้างลิสต์เพลงสำหรับแต่ละงาน แบ่งเป็นเซ็ต 1, 2, 3...

  1. เลือก วันที่ และ สถานที่ ด้านบน
  2. กดเพลงจาก Song Pool (ซ้าย) เพื่อเพิ่มเข้าเซ็ต
  3. ลากเพลง เพื่อจัดลำดับ หรือกดลูกศรขึ้น/ลง
  4. เพิ่มเซ็ตใหม่ด้วยปุ่ม +
  5. กด 💾 บันทึก → เซ็ตลิสต์พร้อมใช้ใน Live Mode
💡 Quota Bar แสดงจำนวนเพลงที่ใช้/ที่กำหนด เพื่อดูว่าเซ็ตเต็มหรือยัง + Duration Bar แสดงเวลารวม

🎤 Live Mode — หน้าจอสำหรับเวที

หน้าจอโหมดมืดเต็มจอ ออกแบบมาใช้ขณะแสดงดนตรีสดบนเวที

ฟีเจอร์หลัก

  • เปลี่ยนเพลง — กดลูกศร ◀ ▶ หรือปัดนิ้วซ้าย-ขวา
  • ข้าม/เพิ่มเพลง — ปัดซ้าย = ลบ, ปัดขวา = เพิ่ม หรือใช้ Context Menu
  • Transpose คีย์ — กดที่กล่องคีย์เพื่อเลื่อนคีย์ขึ้น/ลง
  • Metronome — เปิดจังหวะนับ + เปิด/ปิดเสียง
  • ขอเพลง — แถบด้านล่าง พิมพ์ชื่อเพลง (autocomplete) แล้วส่ง
  • ปรับขนาดตัวอักษร — ปุ่ม A+/A- ด้านบนขวา
  • Badges — 🟣 ขอเพลง 🔵 อังกอร์ 🟡 วันเกิด
⚠️ ครั้งแรกที่เปิด Live Mode จะมี Gesture Tutorial แสดงวิธีใช้ท่าทางต่างๆ สามารถปิดได้หลังดูเสร็จ

⏰ ลงเวลาเข้างาน

  1. เลือก วันที่ (เลื่อนซ้าย-ขวา หรือเลือกจาก date picker)
  2. เลือก สถานที่ จาก dropdown
  3. เลือก รอบเวลา ที่ทำงาน (อิงตามตารางที่สร้างไว้)
  4. กด ✅ ยืนยันลงเวลา

กรณีพิเศษ

  • 🚫 ลา — เลือก "ลาทั้งวัน" หรือ "ลาบางรอบ" + ระบุชื่อคนแทน (ถ้ามี)
  • 🔄 ลงเวลาแทน — เปิด Toggle "ลงเวลาแทนคนอื่น" กรอกชื่อ + เบอร์ติดต่อ
  • ✏️ แก้ไข — หากลงเวลาแล้วสามารถกดแก้ไขหรือยกเลิกได้

📅 ตารางงาน

ดูตารางงานแบบรายสัปดาห์ แต่ละวันแสดง slot ที่กำหนดไว้ (เวลา สถานที่ สมาชิก)

  • 📅 ตาราง Weekly — Grid 7 คอลัมน์ (อา.-ส.) วันนี้หลักจะเด่นด้วยขอบสีทอง
  • 📊 สรุปสถิติ — งานทั้งหมด, งานปกติ, งานนอก, รายได้รวม
  • 🔍 ตัวกรอง — ทั้งวง/รายคน + ช่วงเวลา (รายวัน/สัปดาห์/ย้อนหลัง)
  • ➕ เพิ่มงานนอก (ผู้จัดการ) — กรอกวันที่ เวลา สถานที่ ค่าตัว สมาชิก

📊 รายได้รวม

สรุปรายได้และชั่วโมงทำงาน พร้อมกราฟ 4 แบบ

  • Tabs ช่วงเวลา — รายวัน / สัปดาห์ / เดือน / ปี + ลูกศรเลื่อน
  • กราฟ — 📈 รายได้, ⏱️ ชั่วโมง, 🚫 วันลา, 🔄 ค่าจ้างคนแทน
  • รายละเอียดรายวัน — ตาราง (วันที่, สถานะ, เบรค, ชม., รายได้, คนแทน)
  • ผู้จัดการ — สามารถเลือกดูสมาชิกเฉพาะคน + ดูสรุปทุกคน

🏦 กองกลางวง

3 แท็บ

  • 📒 บัญชีรายการ — ตาราง Ledger (วันที่ ประเภท หมวด รายละเอียด ผู้บันทึก เงิน ยอดสะสม) + กรองตามเดือน/ประเภท
  • ⏳ รออนุมัติ — รายการที่สมาชิกบันทึก รอผู้จัดการอนุมัติ/ปฏิเสธ
  • ➕ บันทึกรายการ — เลือกรายรับ/รายจ่าย กรอกจำนวนเงิน วันที่ หมวดหมู่
💡 สมาชิกทั่วไปบันทึกรายการได้ แต่ต้องรอผู้จัดการอนุมัติก่อนยอดจะเข้าบัญชี

🎸 อุปกรณ์วง

ติดตามอุปกรณ์ทุกชิ้น: เครื่องดนตรี PA แสง และอื่นๆ

  • Tabs กรอง — ทั้งหมด / 🎸 เครื่องดนตรี / 🔊 เสียง-PA / 💡 แสง / 🎒 เสริม / 📦 อื่นๆ
  • Card แต่ละชิ้น — รูปภาพ ชื่อ สถานะ (ปกติ🟢 / ซ่อม🟡 / เสียหาย🔴) ราคา
  • เพิ่มอุปกรณ์ — อัปโหลดรูป กรอกชื่อ ประเภท ราคา Serial No. เจ้าของ ฯลฯ

👥 ข้อมูลวงดนตรี

  • Header วง — ชื่อวง จังหวัด ผู้จัดการ สถิติ (จำนวนสมาชิก, จำนวนร้าน, slots/สัปดาห์)
  • 🏪 ร้านที่เล่น — Card แต่ละร้าน: ชื่อ ที่อยู่ + ตาราง (วัน เวลา สมาชิก)
  • 👤 สมาชิก — ค้นหา/กรอง → Card แสดงชื่อ ตำแหน่ง เบอร์ อีเมล สถานะ

👤 ข้อมูลส่วนตัว

แก้ไขโปรไฟล์และตั้งค่าช่องทางรับเงิน

  • 📝 ข้อมูลส่วนตัว — คำนำหน้า ชื่อ นามสกุล ชื่อเล่น ตำแหน่ง เบอร์โทร
  • 💳 ช่องทางรับเงิน — พร้อมเพย์ ทรูมันนี่ ธนาคาร 7 แห่ง (กสิกร ไทยพาณิชย์ กรุงเทพ กรุงไทย กรุงศรี ทหารไทยธนชาต ออมสิน) + เลขบัญชี
  • 🪪 ข้อมูลบัตรประชาชน — วันเกิด ที่อยู่ (เข้าถึงผ่าน PIN)

💳 เบิกจ่าย & ลงเวลา (ผู้จัดการ)

  1. ตั้งค่าประเภท (รายวัน/สัปดาห์/เดือน) + ช่วงวันที่ + สถานที่ → กด ▶ โหลดตาราง
  2. ดู ตารางเข้างาน — ✅ confirmed, ⏳ pending, █ อยู่ในตารางแต่ยังไม่ลงเวลา
  3. ดู ตารางจ่ายเงิน — คำนวณจาก attendance × อัตราค่าจ้าง
  4. กด 🧾 ออกใบเบิกร้าน หรือ 🧾 ใบแจ้งจ่ายรายคน เพื่อสร้างใบเสร็จ

💵 เบิกจ่ายงานนอก (ผู้จัดการ)

Tab 1: 🎤 งานนอก

  • เลือกงานจากรายการ → ดูสรุปค่าตัว (ค่าตัวรวม ต่อคน ค่าใช้จ่าย กำไร)
  • กด จ่าย ที่สมาชิกแต่ละคน → เลือกวิธีจ่าย (โอน/เงินสด/พร้อมเพย์)

Tab 2: 👤 จ่ายคนนอกวง

  • กรอกชื่อผู้รับ ประเภท (นักดนตรีเกสต์/นักร้อง/เทคนิเชียน) ค่าตัว
  • ดูประวัติการจ่ายด้านล่าง

📁 ประวัติงานนอก (ผู้จัดการ)

  • Stat Chips — งานทั้งหมด / จ่ายครบ / จ่ายบางส่วน / รอเบิก
  • ค้นหา & กรอง — ชื่องาน, สถานที่, สถานะ, ปี, เดือน
  • Card งาน — วันที่ ชื่องาน สถานะ ค่าตัวรวม + กดเปิดรายละเอียด

🧮 คำนวณราคางาน (ผู้จัดการ)

คำนวณราคารับงานนอก แบ่งเป็น 4 ส่วน:

  1. ข้อมูลงาน — ชื่องาน ลูกค้า เบอร์โทร Line วัน/เวลา สถานที่
  2. สมาชิกที่ร่วม — เลือกคนเล่น + ตั้งค่าตัว (เท่ากัน/กำหนดเอง)
  3. รายละเอียด — ชั่วโมง จำนวนรอบ เวลาพัก หมายเหตุ
  4. การเดินทาง — ต้นทาง ปลายทาง ค่าน้ำมัน ยานพาหนะ + ค่าใช้จ่ายเพิ่มเติม

ด้านขวาแสดง Summary Panel สรุปยอดรวมแบบ real-time → กดส่งต่อไปหน้าใบเสนอราคาได้เลย

📄 ใบเสนอราคา (ผู้จัดการ)

  1. กด + ใบเสนอราคาใหม่
  2. กรอกข้อมูลลูกค้า สมาชิก วัน/เวลา สถานที่
  3. เพิ่มรายการ (description / จำนวน / ราคา) — เพิ่ม/ลบแถวได้
  4. ตั้งส่วนลด VAT เงื่อนไขชำระ สถานะ
  5. กด 🖨️ พิมพ์ หรือ 📄 PDF เพื่อสร้างเอกสาร

📜 สัญญาจ้าง (ผู้จัดการ)

  • 📋 ข้อมูลทั่วไป — วันที่ สถานที่
  • 🎤 ฝ่ายผู้ว่าจ้าง — ชื่อ ที่อยู่ เลขบัตร เบอร์โทร
  • 🎸 ฝ่ายผู้รับจ้าง — ชื่อวง ตัวแทน ที่อยู่
  • 🎵 รายละเอียดงาน — ชื่องาน วัน/เวลา สถานที่ จำนวนสมาชิก
  • 💰 การชำระ — ยอดรวม มัดจำ งวดถัดไป วิธีจ่าย บัญชีธนาคาร

❓ คำถามที่พบบ่อย

ข้อมูลจะหายไหมถ้าล็อกเอาท์?

ไม่หาย! ข้อมูลทั้งหมดบันทึกบน Cloud (Supabase) ตลอดเวลา ล็อกอินใหม่ก็เห็นเหมือนเดิม

ติดตั้งเป็นแอปบนมือถือได้ไหม?

ได้! เปิดเว็บใน Safari (iOS) หรือ Chrome (Android) → กดเมนู → "เพิ่มลงหน้าจอหลัก" หรือ "ติดตั้งแอป"

สมาชิกใหม่จะเข้าวงอย่างไร?

ผู้จัดการไปที่ ตั้งค่าวง → คัดลอก รหัสวง 6 หลัก → ส่งให้สมาชิกใหม่ → สมาชิกใหม่ไปสมัครที่หน้า "สมัครสมาชิก" แล้วกรอกรหัสวง

ใช้งานฟรีได้ไหม?

ได้! แพ็กเกจ Free ใช้งานได้ทุกฟีเจอร์หลัก แต่มีโฆษณา 30 วินาทีทุก 75 นาที อัปเกรดเป็น Lite/Pro เพื่อปิดโฆษณาและปลดล็อกฟีเจอร์เพิ่มเติม

ลืมรหัสผ่าน ทำอย่างไร?

หน้าล็อกอิน → กด "ลืมรหัสผ่าน?" → กรอกอีเมล → ตรวจสอบกล่องจดหมาย → กดลิงก์รีเซ็ต → ตั้งรหัสใหม่

จะเปลี่ยน Role สมาชิกได้อย่างไร?

ต้องให้ admin ไปที่ Admin Panel → จัดการผู้ใช้งาน → ค้นหาสมาชิก → เปลี่ยน Role

เปิดจาก LINE/Facebook แล้วใช้งานไม่ได้?

In-App Browser มีข้อจำกัด → กดเมนู "เปิดใน Safari/Chrome" ระบบจะแสดง Banner แนะนำให้อัตโนมัติ
